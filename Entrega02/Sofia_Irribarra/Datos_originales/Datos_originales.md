## Base de datos originales

No existe una base de datos única y consolidada que reúna la información completa sobre mujeres que han ocupado cargos ministeriales en Chile desde 1952 hasta la actualidad. Por esta razón, la base utilizada en este trabajo fue construida mediante un proceso de recopilación, sistematización y cruce de información proveniente de distintas fuentes disponibles en línea.

La construcción de esta base implicó no solo la recopilación de datos administrativos (como nombres, años y ministerios), sino también un proceso de enriquecimiento con variables biográficas, lo que requirió consultar múltiples registros individuales.

A continuación, se presenta el listado de fuentes utilizadas:

- Wikipedia – Lista de ministras de Chile:

- Biblioteca del Congreso Nacional (BCN):  
https://www.bcn.cl/portal/
